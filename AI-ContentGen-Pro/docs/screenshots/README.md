## How to Create Screenshots

1. **Start the application**:
   ```bash
   python gui/app.py
   ```

2. **Open browser** to http://127.0.0.1:5000

3. **For each screenshot**:
   - Set up the desired view
   - Press `Windows + Shift + S` (Windows) or `Cmd + Shift + 4` (Mac)
   - Select the area to capture
   - Save the screenshot to this directory

4. **Recommended dimensions**: 1200x800px or larger

5. **File format**: PNG with transparency if possible

## Tips for Great Screenshots

- Use a clean browser window (close unnecessary tabs)
- Fill in example data that looks realistic
- Ensure fonts are readable
- Use consistent browser zoom level (100%)
- Hide personal/sensitive information
- Use high-quality/retina display if available
