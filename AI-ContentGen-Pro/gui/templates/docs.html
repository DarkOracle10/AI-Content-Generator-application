API Overview

The AI Content Generator API provides programmatic access to content generation capabilities. All API endpoints return JSON responses.

Base URL
http://localhost:5000/api
Response Format

All responses follow this structure:

{
  "success": true,
  "data": { ... }
}

// Or on error:
{
  "success": false,
  "error": "Error message",
  "code": "ERROR_CODE"
}

Authentication

The API uses session-based authentication by default. For programmatic access, you can optionally configure API key authentication.

Optional API Key

If API_KEY_REQUIRED environment variable is set, include the key in headers:

X-API-Key: your-api-key
Rate Limiting

Default limit: 60 requests per minute per session.

GET /api/templates

List all available content templates.

Query Parameters
NameTypeDescription
category string Filter by category (marketing, technical, etc.)
Response
{
  "success": true,
  "templates": [
    {
      "name": "product_description",
      "category": "marketing",
      "description": "Generate product descriptions",
      "required_variables": ["product_name", "features", "audience"],
      "version": "1.0"
    }
  ],
  "count": 10
}
POST /api/generate

Generate content using a template.

Request Body
NameTypeDescription
template * string Template name
variables * object Template variables
use_cache boolean Use cached results (default: true)
options object Generation options (temperature, max_tokens)
Example Request
POST /api/generate
Content-Type: application/json

{
  "template": "product_description",
  "variables": {
    "product_name": "Smart Watch Pro",
    "features": "GPS, Heart Rate Monitor, 7-day battery",
    "audience": "Fitness enthusiasts"
  },
  "use_cache": true,
  "options": {
    "temperature": 0.7
  }
}
Response
{
  "success": true,
  "result": {
    "content": "Introducing the Smart Watch Pro...",
    "template_used": "product_description",
    "tokens_used": {
      "prompt": 45,
      "completion": 123,
      "total": 168
    },
    "cost": 0.0034,
    "timestamp": "2026-01-05T21:35:00Z",
    "cached": false,
    "generation_time": 2.3
  }
}
POST /api/generate/variations

Generate multiple variations of content.

Request Body
{
  "template": "social_media_post",
  "variables": {
    "topic": "AI Innovation",
    "platform": "LinkedIn"
  },
  "count": 3,
  "temperature_range": [0.5, 1.0]
}
Response
{
  "success": true,
  "variations": [
    {"content": "...", "variation_number": 1, "variation_temperature": 0.5, ...},
    {"content": "...", "variation_number": 2, "variation_temperature": 0.75, ...},
    {"content": "...", "variation_number": 3, "variation_temperature": 1.0, ...}
  ],
  "total_cost": 0.012,
  "success_count": 3,
  "failure_count": 0
}
POST /api/generate/batch

Process multiple generation requests in one call.

Request Body
{
  "requests": [
    {"template": "product_description", "variables": {...}},
    {"template": "meta_description", "variables": {...}}
  ],
  "parallel": false
}
Response
{
  "success": true,
  "results": [...],
  "total_cost": 0.025,
  "success_count": 2,
  "failure_count": 0
}
GET /api/history

Get generation history for the current session.

Query Parameters
NameTypeDescription
limitintNumber of items (default: 20, max: 100)
templatestringFilter by template name
sinceISO dateItems after this date
GET /api/history/export

Export history as downloadable file.

Query: ?format=json or ?format=csv

DELETE /api/history/clear

Clear all generation history.

POST /api/validate

Validate template variables without generating content.

Request
{
  "template": "product_description",
  "variables": {"product_name": "Test"}
}
Response
{
  "success": true,
  "valid": false,
  "missing_variables": ["features", "audience"],
  "estimated_cost": null
}
POST /api/cost-estimate

Estimate cost before generating content.

Request
{
  "template": "product_description",
  "variables": {
    "product_name": "Widget",
    "features": "Fast, reliable",
    "audience": "Developers"
  }
}
Response
{
  "success": true,
  "estimate": {
    "input_tokens": 45,
    "estimated_output_tokens": 150,
    "total_tokens": 195,
    "estimated_cost": 0.0039,
    "model": "gpt-4o-mini"
  }
}
